Now you know how to get your program to do maths operations, you should learn how to get information off your user.
That way your program won't be a one trick pony!

Scanners

Sometimes you need to get some information from something. Maybe it's a file, or perhaps it's user input. Either way you need some code that will read this into your program. Then you can do something with it.
Enter the Scanner class! A Scanner will take in a stream of data (basically where data is progressively arriving) at the program, such as a user typing slowly, and then give the result to your code in a useful format.

In this example, we will be creating a scanner to take user input. 

        
          Scannner myScanner = new Scanner(System.In);
This creates a new Scanner called myScanner, and creates it using System.In, which is the input on the command line. This will make more sense a bit later.

So, how does this help us? Now that we have a Scanner we can pull some user input in. Enter the following code, and see what happens when you run it. 

        
          Scanner myScanner = new Scanner(System.In);
          System.out.println("Please enter your name:");
          String name = myScanner.nextLine();
          System.out.println("Nice to meet you, " + name + "!");
