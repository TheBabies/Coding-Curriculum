Functions and beyond!

Till now we tried to show basic operations and statement s in java, but there is a way that you can combine couple of statments and build something called a function. Functions in programming are pretty much similar to functions in maths. it is basically like a machine which gets something as input, does some changes on it based on the built-in statements in it, and gives the result as an output! quite straightforward, isn't it? let's go back to finding the absolute of a number's problem. if you combine all of the statements as below, then you will have a function named absolute which can get any integer number and give the absolute of that number back. 

        
          int absolute(int a)
          {
            if(a>=0)
            {
              return a;
            }
            else
            {
              return a*-1;
            }
          }
You can see that the keyword "int" is written before ur function name. That is the type of value your function will give as output, or in coding terminology it returns If you return any other value than an integer in the function, you will face an error from your computer! also, after the function name (which here is absolute), you can see a paranthesis which has a declared variable inside. this is where you say how many inputs you have, and from what type, and in your function with what name they are called(so that you can do any operation on them). after all these declaration, you will have a block of curly brackets, where you actually write the statements of your function. and that is it! The point here is that anytime in your code that you need, you can simply send any integer value to this function and get your result back. it is referred to as calling a function. for example, after declaring the absolute function, you can use it like this: 
        
          System.out.println(absolute(3));
          System.out.println(absolute(-5));
          int a = absolute(-7);
          System.out.println(a);
this is the basics of how it works :) try it out in your own IDE and see how it works...
