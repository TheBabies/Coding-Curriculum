What is Programming?


Programmers looking happy because code is cool.

Computer programming is all around us. Right now you are using many programs, all made by people like you. The idea is that you tell the computer what to do using coded instructions. From this you can start with the most basic of software and over time you can build up to more complicated, larger projects. If you are going to talk to another person you need to know a language that they know. Computers work in the same way, except they cannot speak English. Your job as a programmer is to learn a language that a computer can understand, and that's what this website will teach you.

When you write code in a language like Java it is translated into patterns of 1s and 0s that the computer can understand. Each group of 1s and 0s is called an 'instruction' that your computer's Processor chip can follow. Obviously 1s and 0s are far too complicated for even the best programmers to work with, so a programming language is used that's close enough to English to read and write, but organised enough for a computer to use.

It's really important to remember that without programmers there would be no programs, and without programs your computer would pretty much be useless! The world does not have enough programmers so we need more people like you to learn this stuff.

What will I be learning?

In this tutorial you will be learning a programming language called Java. Java is important because it works on all computers (Windows, Mac and Linux), and is needed if you ever want to build apps for Android phones. Almost every software company uses Java sometimes. Also, it's a really nice language for beginners (like you!) to start with because a lot of the difficult memory things are done for you, so you can focus on telling the computer what to do and not how to do it.

Why should I bother?

If you are interested in how the technology in your life works or you want technology to do something new you must learn to program. Also if you get good at it there are a lot of very well paid programming jobs that you can get. Programming is a really enjoyable life skill that will help you to think more intelligently whilst stretching your brain.
